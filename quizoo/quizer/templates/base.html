ABOUT

Quizoo is a project created with close reference to the current academic situation of schools and colleges due to the Covid-19 pandemic. We have have found that the current available platforms are not yet ready for conducting examinations as they crash or run into errors minutes before the it starts.

With Quizoo, creating tests becomes easy. It is a great tool and extremely user friendly. Teachers can create objective type questions, set time limits, submission dates etc.
